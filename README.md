# Requirements
- Python 3.13.0

# Activate virtual environment
```bash
pipenv shell
```

# Install dependencies
```bash
pipenv install
```

# Run the application
```bash
python application.py
```

# Run unit tests
```bash
python -m unittest
```

# Simulate a brute force attack
```bash
python brute_force.py
```

# Run flake8
```bash
flake8
```
